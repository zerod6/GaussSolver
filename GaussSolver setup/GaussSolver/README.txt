Приложение GaussSolver

	Приложение GaussSolver предназначено для решения систем линейных уравнений методом Гаусса с частичным выбором главного элемента.
 
	Если введена квадратная система (кол-во переменных=кол-во уравнений), она будет решаться по методу Гаусса с частичным выбором главного элемента. Однако, если система является несовместной (кол-во переменных>кол-во уравнений), программа получит решение методом наименьших квадратов. Если система переполнена (кол-во уравнений>кол-во переменных), то решений либо не будет, либо их будет бесконечно много. 

Как задать значения:
	1. Открыть заранее подготовленный файл с значениями. Он должен быть в формате txt, каждое новое уравнение находится на новой строке, числа, используемые как значения переменных, следует писать через пробел. Программа работает с целыми и дробными числами, как положительными, так и отрицательными;
	2. Ввести значения с клавиатуры. Чтобы ввести значение, достаточно нажать на нужную ячейку таблицы левой кнопкой мыши и ввести значение с клавиатуры. Перемещаться межу ячейками можно при помощи стрелок на клавиатуре, а также нажатием на левую кнопку мыши.
Обратите внимание, что программа работает только с системами, не выходящими за следующие ограничения: количество уравнений не больше 1000, количество переменных и знаков после запятой не больше 100. Минимальные значения: 2 для количества переменных и уравнений, 0 для количества знаков после запятой.
	Если переменных/уравнений слишком много, и они не помещаются в таблице, перемещайтесь по ней с помощью появившегося ползунка. 

Как поменять параметры:
	1. Количество переменных. Поле расположено в левой верхней части экрана, в него можно как самостоятельно вводить значения с клавиатуры, так и использовать кнопки для увеличения/уменьшения количества на 1;
	2. Количество уравнений. В левой верхней части экрана, под полем с количеством переменных, располагается количество уравнений. В него можно записывать значения с клавиатуры, а также менять их на 1 кнопками;
	3. Количество знаков после запятой. Поле расположено в правой верхней части экрана. Количество можно ввести с клавиатуры, либо увеличивать/уменьшать на 1 с помощью кнопок. Если вы уже получили результат, но хотите изменить отображающееся количество значений после запятой, выберите в поле новое значение и нажмите кнопку «Решить».
	Обратите внимание, что при добавлении переменных/уравнений, уже введенные данные останутся, и к ним добавятся пустые ячейки. Но если уменьшить количество переменных/уравнений, то данные, находящиеся в удаленных столбцах/строках безвозвратно удалятся. 

Как получить результат:
	После нажатия кнопки «Результат» значения переменных появятся в правой части экрана. Оттуда их можно скопировать, либо сохранить на компьютер в текстовый файл. Для этого нажмите кнопку «Сохранить результат», выберите место для создания файла либо вручную, либо введите путь к нему, а также дайте файлу имя. В файле запишется результат в таком же виде, как он представлен на экране. 
	Если вы получили значение переменной «не число», это значит, что либо система несовместна и не имеет решений, либо система имеет бесконечное количество решений.
	После получения результата можно распечатать его на принтере. Для этого нажмите кнопку «Распечатать результат» и выберите принтер, а также задайте нужные параметры.
